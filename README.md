# AISchedule-IMAU

为小爱课程表适配内蒙古农业大学教务系统（通过统一身份认证服务登录）。

## 使用方式

- MIUI 系统日历中订阅课程表服务并打开。
    ![](assets/step-0.jpg)

- 设置中选择学校：内蒙古农业大学-统一身份认证服务。
    ![](assets/step-1.jpg)

- 课表页点击教务系统导入。
    ![](assets/step-2.jpg)

- 登录后打开教务系统点击个人课表并点击一键导入。
    ![](assets/step-3.jpg)
